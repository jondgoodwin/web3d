Quat Type

Quat is a quaternion whose x,y,z,w floating point components specify a rotational orientation. The x,y,z components represent the axis of rotation. The w component determines the rotation around that axis.

Class Properties (see Type)

New
(Quat) or (x,y,z,w)-> quat Create a new Quat value. Its x,y,z,w components are initialized by another Quat or up to 4 Floats as parameters. Otherwise, the components are initialized to 0.0, 0.0, 0.0, 1.0.

Instance Methods

Conjugate
(Quat) or ()-> Quat Return a Quat's conjugate. If an Quat value is passed, put its conjugate in self. Otherwise calculate self's conjugate into self.
Inverse
(Quat) or ()-> Quat Inverse and return an Quat value. The inverse represents the opposite rotation, achieved by normalizing its conjugate. If an Quat value is passed, put its inverse in self. Otherwise invert self.
Lerp
(Xyz1, Xyz2, scalar) -> xyz. Calculate and return Xyz value linearly interpolated between two Xyz values by a scalar from 0 to 1.
Mult
Multiply two quaternions, returning result in self. Either multiplies self by parm1 or parm1 * parm2
Normalize
-> Quat Normalize and return self to its unit equivalent (length is 1.0).
Set
(Quat) or (x,y,z,w)-> quat Update Quat value's x,y,z,w with components from passed Quat or passed x,y,z,w Floats
SetAngleAxis
(angle, axisx, axisy, axisz) or (Float, Xyz)-> quat Set Quat's x,y,z,w to correspond with rotation axis and angle (right-hand rule).
SetRotateXyz
(x,y,z) or (Xyz)-> quat Update Quat's x,y,z,w to correspond to Euler axis rotations, first X then Y then Z.
SetRotateYxz
(x,y,z) or (Xyz)-> quat Update Quat's x,y,z,w to correspond to Euler axis rotations, first Y then X then Z.
SetVectors
(Xyz,Xyz)-> quat Calculate and return a Quat able to rotate a vector from u vector to v vector
Slerp
(quat1, quat2, scalar) -> quat. Calculate and return self's quat value via spherical, linear interpolation between two Quat values using a scalar from 0 to 1.
