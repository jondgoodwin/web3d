Buffer Type

Buffer contains a collection of numbers, all of the same type. Typically, these hold a Shape's vertex information, such as position (Xyz) or color. For example:

position: +Buffer(Xyz, 20)
	1.0, 0.0, 2.5
	2.0, 0.4, 2.4 # ...

Class Properties (see Type)

New
(itemType, size)-> buffer. Create a new buffer containing space for size items of itemType (e.g., Xyz).

The itemType should be the class constructor for that type. itemType's instance type must have the .itemcount and .varspace properties.

_newtype
-> mixin. The type used by buffer instances.

Instance Properties

<<
(value)-> self. Append a single value or item of the declared type. If needed, this will grow the size of the buffer.
